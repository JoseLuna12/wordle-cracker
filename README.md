# wordle-cracker

### use
```
cargo run <lang> <action> <input>
```

#### lang
suported languages:
- spanish (es)
- english (en)

#### action
- help
- firstLetter
- lastLetter
- contains
- contain
- incorrectWords
- pattern
- staticLetters

#### input
- <a>
- <a,b,c,d>
- <hello,words,test>
- <_e_l_> (pattern)
